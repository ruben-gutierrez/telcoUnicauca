Acceder al núcleo Clearwater IMS todo en uno (AIO)

Para acceder al núcleo Clearwater IMS es necesario reservar una arquitectura e ingresar a la consola de configuración de la máquina virtual Clearwater (aio) por medio del botón de “consola”. Las credenciales de ingreso de la máquina virtual (aio) son: usuario: "ubuntu"; contraseña: "cw-aio".

Terminal de comandos máquina virtual AIO

Cuando se presione el botón de consola se abrirá un terminal de comandos de la MV en la web para su posterior configuración. En esta pestaña aparecerá un cuadro negro que representa el terminal de comandos de la MV. Para poder configurarlo, es necesario presionar la tecla “enter” sobre el cuadro negro permitiendo al usuario ingresar a la MV como modo privilegiado. Para acceder como modo “súper usuario” debe escribir en el terminal el comando "sudo su" e ingresar las credenciales mencionadas anteriormente.

Acceder a la Dashboard de Clearwater IMS

Cuando se accede al terminal de comandos de la máquina virtual Clearwater AIO es necesario verificar que los servicios del núcleo IMS estén subidos mediante el comando ”monit summary”. Posteriormente, se debe ingresar a la Dashboard de Clearwater IMS para configurar los perfiles de usuario que serán utilizados en los servicios de llamada y videollamada de IMS. Para ello, se debe copiar la IP pública de la MV de AIO en la URL del navegador como se observa en la Figura 4 la cual permite ingresar a la Dashboard de Clearwater IMS

Crear un usuario administrador de Clearwater

Para administrar los servicios del núcleo Clearwater IMS es necesario crear un usuario administrador que permita gestionar los perfiles de usuario del núcleo IMS para acceder a sus servicios. En la opción or signup visualizada en la Figura 4 se ingresa a la página de crear usuario. Posteriormente, en la Figura 5, se muestra un ejemplo crear de un usuario. Nota: en la opción “signup code” debe ir siempre la palabra “secret”

Crear un perfil de usuario en Clearwater

En esta opción se configuran los perfiles de usuario para que los usuarios puedan autenticarse y acceder a los servicios del núcleo Clearwater IMS. En la Figura 6 se observa un ejemplo de una cuenta creada que será utilizada para configurar un softphone. En este caso, el usuario a configurar tendrá el perfil de usuario: 6505550776@example.com y la contraseña: QDeRTDV4k.

Los perfiles de usuario pueden ser modificados por el usuario administrador quien tiene opciones tales como: resetear contraseñas, crear usuarios, eliminar usuarios o configurar permisos.

Configurar un softphone

El softphone es una herramienta que permite acceder a los servicios del núcleo Clearwater IMS tales como llamada o videollamada. Asimismo, permite acceder a los servicios de otros sistemas de Telecomunicaciones como Asterisk. En este caso, el softphone utilizado es Zoiper el cual debe ser configurado con los parámetros de cada perfil de usuario creado en el núcleo IMS de Clearwater. En la Figura 6, se observan las credenciales de un perfil de usuario (usuario: 6505550776@example.com ; contraseña QDeRTDV4k) utilizado para configurar el softphone visualizado en la Figura 7.


Una vez configurado el softphone es necesario verificar si se registró correctamente. Validado el registro puede probar los servicios de IMS tales como llamada y videollamada. Nota: Es importante tener configurado mínimo 2 equipos softphone.
