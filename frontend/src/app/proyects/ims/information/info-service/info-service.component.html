SERVICIOS PLATAFORMA TELCO

La plataforma Telco provee servicios de orquestación y gestión de recursos virtuales instanciados en arquitecturas de máquinas virtuales como el núcleo IMS. En particular, el proyecto Telco IMS provee los servicios descritos a continuacion:

  • Máquinas Virtuales (VM): Las VM son orquestadas por el hypervisor de Openstack que es un sistema de virtualización de funciones de red que permite la gestión e instanciación de funciones de red alojadas en arquitecturas de VM como el subsistema multimedia IP (IMS). IMS provee una arquitectura para la investigación, desarrollo tecnológico e innovación de sistemas de Telecomunicaciones convergentes sobre IP. La plataforma Telco IMS provee a los usuarios un un ambiente de pruebas flexible y configurable mediante el acceso a las máquinas virtuales por medio de un terminal de consola en la web.

    Para acceder a las arquitecturas de VM como el núcleo IMS o crear un nueva máquina virtual, es necesario reservar una arquitectura. Para mayor información puedes acceder a las guías de usuario

  • Pruebas: El proyecto Telco IMS provee a los usuarios una guía práctica para la experimentacion de pruebas en el nucleo IMS como la configuración y simulación de usuarios y servidores proxy SIP que permitan evaluar el rendimiento y QoS del sistema por medio de la monitorización de recursos en las máquinas virtuales.
  • Gráficas de Monitorización: Este sistema provee gráficas de monitorización de recursos de procesamiento y almacenamiento de las arquitecturas de VM asignadas a cada usuario en el sistema. Las gráficas permiten evaluar el rendimiento y QoS de las arquitecturas IMS cuyos datos pueden ser descargados en formato JSON para su posterior analisis.
